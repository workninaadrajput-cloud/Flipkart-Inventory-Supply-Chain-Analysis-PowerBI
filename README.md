# 📦 Flipkart Inventory & Supply Chain Analysis Dashboard | Power BI

## 📖 Project Overview

This Power BI project analyzes Flipkart's inventory and supply chain operations to monitor stock performance, warehouse utilization, transportation costs, inventory levels, lead times, and order fulfillment efficiency.

The dashboard provides actionable insights that help businesses optimize inventory management, improve supply chain performance, and support data-driven decision-making.

---

## 🎯 Business Objectives

- Monitor warehouse utilization and inventory efficiency.
- Analyze transportation costs across regions and product categories.
- Track inventory levels by region and category.
- Evaluate lead times across product categories.
- Monitor sales trends and units sold over time.
- Identify backorders and order fulfillment issues.

## 📌 Key Performance Indicators (KPIs)

| KPI | Description |
|------|-------------|
| Warehouse Utilization | Measures warehouse capacity usage |
| Days Sales of Inventory | Indicates inventory holding period |
| Inventory Turnover Ratio | Measures inventory movement efficiency |
| Average Lead Time | Time taken to replenish inventory |
| Transportation Cost | Logistics expenditure analysis |
| Backorder Count | Delayed or unfulfilled orders |
